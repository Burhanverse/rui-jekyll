With RUi in hand you can experience pure MIUI but with the customisations. RemodedUi is currently based on MIUI 12.5/13/14 versions.

Remoded Ui aka RUi is a totally debloated rom with just the core Xiaomi applications that are required by the system.

Stability

What is Remoded Ui?

RUi or Remoded Ui is a modified stock MIUI 13/14 rom which brings tones of customisation options alongwith stablity in mind.

Some Highlighted Features

  • With RUi a user can have almost same experience as MIUI 14.
  • Most of the cutting edge features are ported from MIUI 14.
  • RUi is saftynet passed by default, which means you can use banking apps on the rom without root.
